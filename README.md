# Eli Cordeiro Doces - Site Oficial

Bem-vindo ao repositório do site da **Eli Cordeiro Doces**, uma doceria e confeitaria artesanal. Este projeto foi criado com foco em performance, design moderno e experiência visual atraente, mostrando os produtos e facilitando pedidos e contato via WhatsApp.

---

## 🛠 Tecnologias utilizadas

- **[Vite](https://vitejs.dev/)**
- **[React](https://reactjs.org/)** 
- **[TypeScript](https://www.typescriptlang.org/)**
- **[Tailwind CSS](https://tailwindcss.com/)** 
- **[shadcn-ui](https://shadcn-ui.com/)**

## 🚀 Como rodar o projeto localmente

1. Clone o repositório:

git clone <REPO_URL>
cd eli-cordeiro-doces

2. Instale as dependências:

npm install

3. Rode o servidor de desenvolvimento:

npm run dev

4. Acesse o site no navegador:

http://localhost:5173

## 🎨 Estilização

O projeto utiliza Tailwind CSS junto com shadcn-ui para componentes prontos, responsivos e acessíveis.

O design foca em:

- Galeria de doces visual e interativa
- CTA para WhatsApp integrado para pedidos e informações
- Layout responsivo para mobile e desktop
